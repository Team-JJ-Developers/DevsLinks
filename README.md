# DevLinks 🚀


Uma landing page minimalista estilo "Link in Bio", desenvolvida especialmente para centralizar portfólios, redes profissionais e canais de orçamento para desenvolvimento web.

## 📌 Sobre o Projeto

Este projeto funciona como um cartão de visitas digital otimizado para dispositivos móveis. Ele facilita o contato de potenciais clientes interessados na criação de Sites, Landing Pages e Bots, além de direcionar recrutadores e parceiros para os perfis do GitHub e LinkedIn dos desenvolvedores.

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído com foco em performance e estilização rápida, utilizando:

* **React:** Biblioteca JavaScript para construção da interface.
* **Tailwind CSS:** Framework utilitário para a estilização responsiva e tema escuro.

## ✨ Funcionalidades

* **Call to Action (CTA) de Orçamentos:** Botões com destaque visual para direcionar clientes diretamente para o contato de vendas.
* **Links Individuais:** Separação clara das redes profissionais (LinkedIn e GitHub) do Frontend e do Backend.
* **Design Responsivo:** Layout adaptável e otimizado para navegação mobile (onde a maioria dos usuários acessa links de bio).
* **Dark Mode Nativo:** Estilização moderna utilizando a paleta escura do Tailwind.
